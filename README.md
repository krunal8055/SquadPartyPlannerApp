# SquadPartyPlannerApp
Squad Party Event App is an android Java based application which is used by user for create or host any event and invite other user of app or non user of app using firebase services like Firebase Realtime Database, Firebase Authentication, Firebase Storage, Firebase Functions, Firebase Cloude Messaging,and Dynamic Links From Firebase.
